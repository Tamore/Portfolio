A high-fidelity, terminal-inspired professional portfolio designed for researchers and engineers. **NT Research Dashboard** combines a "System Console" aesthetic with an editorial layout to showcase expertise in **Distributed Systems**, **Event-Driven Architectures**, and **Cloud Security**.

![Mascot Preview](/public/mascot.png)

## 🌟 Key Features
- **Terminal Journey Log**: An interactive CLI-inspired narrative of research focus and academic foundation.
- **Precision UI**: A custom-crafted design system using a signature "Candy Pink" and "Obsidian Purple" palette.
- **Dynamic Project Console**: High-contrast grid for showcasing complex engineering artifacts.
- **Bespoke Branding**: Features the "Doodle-Saur"—a custom neon-line art mascot representing the intersection of research and development.
- **Glassmorphic Sidebar**: A streamlined "Command Dock" for efficient site-wide navigation.

## 🚀 Research Focus
- **Distributed Systems**: Resilience patterns and fault tolerance in large-scale platforms.
- **Event-Driven Architecture**: Designing secure and reliable cloud-native systems.
- **Observability**: End-to-end failure analysis through logs, metrics, and traces.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📐 Project Structure
- `/src/App.tsx`: Main dashboard architecture and component logic.
- `/src/main.ts`: Application entry point.
- `/public/`: Static assets including the **Doodle-Saur** mascot and professional avatars.
- `DEVELOPMENT.md`: Detailed technical documentation and design philosophy.
